# /blogkit.taskstoissues Command

**Purpose**: Convert tasks.md into actionable, dependency-ordered GitHub issues.

**Usage**: Run `/blogkit.taskstoissues` in Claude Code to create GitHub issues from tasks.

**Status**: PLACEHOLDER - Full implementation in Phase 9 (T081-T082)

This slash command will:
1. Read tasks.md from feature directory
2. Create GitHub issues with dependencies, labels, milestones
3. Maintain task organization (sprints, user stories, checkpoints)
4. Link related issues based on dependencies

**Reference**: See `refs/4_pm_tasking_for_tasks.md` for task structure.
