# /blogkit.tasks Command

**Purpose**: Generate content production tasks organized by sprint and post.

**Usage**: Run `/blogkit.tasks` in Claude Code after plan to break down implementation into actionable tasks.

**Status**: PLACEHOLDER - Full implementation in Phase 7 (T058-T064)

This slash command will:
1. Reference `refs/4_pm_tasking_for_tasks.md` and `refs/5_more/tasking_methodology.md` for task structure
2. Organize tasks by content sprint (Sprint 0: Platform Setup, Sprint 1: Post 1, Sprint 2: Post 2, etc.)
3. Include dependencies (e.g., "Post 1 published" before "Post 2 research begins")
4. Include validation checkpoints (analytics review, SEO ranking check, engagement review)
5. Follow 4-pass editorial workflow per post (research, draft, technical review, final edit)

**Reference**: See `refs/4_pm_tasking_for_tasks.md` for guidance on task structure.
